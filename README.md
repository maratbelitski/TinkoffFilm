# Description

Тестовое задание для компании Tinkoff, финтех-джуниор.
Представляет собой реализацию поиска и ознакомления c фильмами по API.

# Techologies

Язык Kotlin

В ходе работы использовалось:
- XML;
- RecyclerView;
- ListAdapter;
- MVVM;
- JSON objects;
- Kotlin coroutines;
- Retrofit;
- Room;
- RXJava
- а также принципы Clean architecture, SOLID;
